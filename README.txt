МОДУЛЬ КУРАТОР ДЛЯ BITRIX24
============================

ИНСТРУКЦИЯ ПО УСТАНОВКЕ:

1. Распакуйте архив в /bitrix/modules/ так, чтобы получилось:
   /bitrix/modules/custom.curator/

2. Правильная структура:
   /bitrix/modules/custom.curator/
   ├── install/
   │   ├── index.php         ← ВАЖНО!
   │   ├── version.php
   │   ├── db/
   │   │   ├── install.sql
   │   │   └── uninstall.sql
   │   ├── js/
   │   └── css/
   ├── lib/
   ├── include.php
   └── .settings.php

3. Перейдите в администрирование:
   Администрирование → Настройки → Настройки продукта → Модули

4. Найдите "Куратор в задачах" и нажмите "Установить"

ВАЖНО:
- Модуль требует установленный модуль "Задачи" (tasks)
- После установки автоматически создастся таблица b_task_curator

ПРОВЕРКА УСТАНОВКИ:
- В разделе Задачи появится поле "Куратор"
- REST API методы curator.* будут доступны
- Таблица b_task_curator создана в БД
