МОДУЛЬ КУРАТОР ДЛЯ BITRIX24
============================

ИНСТРУКЦИЯ ПО УСТАНОВКЕ:

1. Распакуйте архив так, чтобы папка "custom.curator" была в /bitrix/modules/

2. Структура должна быть:
   /bitrix/modules/custom.curator/
   ├── lib/
   ├── install/
   ├── module.php
   ├── include.php
   ├── .settings.php

3. Перейдите в администрирование Bitrix24:
   Администрирование → Модули и компоненты → Модули

4. Найдите модуль "Куратор в задачах" и нажмите "Установить"

5. После установки модуль автоматически создаст таблицу b_task_curator

ИСПОЛЬЗОВАНИЕ:

- При создании задачи добавьте куратора в поле "Куратор"
- Куратор получает права наблюдателя (может просматривать и комментировать)
- В разделе "Задачи" появится новая вкладка "Куратирую"

REST API МЕТОДЫ:

- curator.add - добавить куратора
- curator.remove - удалить куратора
- curator.list - получить список кураторов
- curator.gettasks - получить задачи пользователя в качестве куратора
- curator.check - проверить, является ли пользователь куратором
